# 基于springboot和vue的选课管理系统9



## 1、项目介绍

基于springboot和vue的选课管理系统9是前后端分离系统

管理员：学生管理、教师管理、成绩管理、课程管理等

学生：选课、退课、查看成绩

教师：开设课程、登记成绩

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：VUE

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：Tomcat 7-10版本均可
- tomcat版本：无需
- maven版本：无限制
- nodejs：14
- 硬件环境：Windows
